# Project Requirements — Anwar Alam Portfolio

## Goal
The best developer portfolio in the UAE market — convinces recruiters, agencies,
and founders that Anwar is a senior developer, and increases interview calls.

## Audience
UAE recruiters · international recruiters · Shopify agencies · software companies ·
startup founders · clients.

## Tech stack
React 19 · Vite · TypeScript · Tailwind CSS v4 · shadcn-style UI · Framer Motion ·
React Router · React Icons · Lucide · GSAP (only where needed).

Cross-cutting: dark/light theme, responsive, SEO-optimized, accessible, lazy
loading, image optimization, reusable components, clean folder structure.

## Performance targets
Lighthouse Performance 95+ · Accessibility 95+ · SEO 100 · Best Practices high.
Code splitting, lazy routes, tree shaking, optimized/lazy images.

## Architecture
Clean, scalable, expandable to blog + admin dashboard + CMS. Small files, one
section per component, reusable hooks, TypeScript everywhere, no duplication.
